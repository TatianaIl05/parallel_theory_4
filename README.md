# Task 4
Цель: чтения данных с датчиков и формирование картинки. Задержки
должны быть минимальны.
Для всех датчиков:
1) данные с датчиков нужно считывать постоянно, для получения самых
последних значений;
2) каждый датчик обрабатывать в отельном потоке;
3) обмен данными между потоками организовать через очереди (модуль
queue).
